Tab X 1.1.0 updates the public product page and documentation around the refreshed Chrome new tab experience.

## What's new

- Presents open tabs in clearer domain groups with quick close and bookmark controls
- Adds visible Saved for later and Archived panels for local tab triage
- Adds sorting controls for tab-heavy sessions
- Keeps top sites, manual shortcuts, the search box, Ganzhi time, and natural-language clock in one calmer view
- Updates the LFree screenshot to match the 1.1.0 interface

## Install

Download the latest source zip from GitHub, unzip it, enable Developer mode in `chrome://extensions`, then reload or load the `extension/` folder as an unpacked extension.
