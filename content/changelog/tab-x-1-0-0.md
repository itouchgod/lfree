Tab X is now listed on LFree as the third public app.

## What's included

- Replaces Chrome's new tab page with a local dashboard
- Groups open tabs by main domain
- Supports top sites, manual shortcuts, and drag-to-save shortcuts
- Flags duplicate URLs and can close extra copies
- Saves tabs for later in `chrome.storage.local`

## Install

Download the source zip from GitHub, unzip it, enable Developer mode in `chrome://extensions`, then load the `extension/` folder as an unpacked extension.
