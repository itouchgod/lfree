Tab X 1.1.0 围绕新版 Chrome 新标签页体验，更新了 LFree 上的产品页面、说明文档和截图。

## 本次更新

- 以更清晰的域名分组展示打开的标签，并提供快速关闭和收藏操作
- 新增可见的稍后再看与已归档面板，用于本地整理标签
- 增加排序控件，方便处理标签较多的浏览会话
- 保留常用站点、手动快捷入口、搜索框、干支时间和英文自然语言时钟
- 将 LFree 展示截图更新为 1.1.0 新界面

## 安装

从 GitHub 下载最新源码 zip，解压后在 `chrome://extensions` 开启开发者模式，然后重新加载或加载其中的 `extension/` 文件夹。
