FileNest 是一款轻量级 macOS 工具，用来集中收藏、打标签并快速打开常用文件夹，可从主窗口或菜单栏进入。所有数据保存在本机。

> **当前版本：v0.1.3** · 已发布

## 下载

根据 Mac 芯片选择安装包：

| Mac 类型 | 安装包 |
|----------|--------|
| **Apple 芯片**（M1 / M2 / M3 / M4） | [FileNest-mac-arm64.zip](https://github.com/itouchgod/filenest/releases/download/v0.1.3/FileNest-mac-arm64.zip) |
| **Intel 芯片** | [FileNest-mac-x64.zip](https://github.com/itouchgod/filenest/releases/download/v0.1.3/FileNest-mac-x64.zip) |

也可在 [GitHub 发布页](https://github.com/itouchgod/filenest/releases/tag/v0.1.3) 下载 DMG 安装镜像。

不确定芯片型号？打开 **关于本机**，查看 **芯片** 或 **处理器**。

也可在 [FileNest 产品页](/zh/apps/filenest) 直接下载。

## 安装

1. 下载并解压 `.zip`（或打开 `.dmg`，将 **FileNest.app** 拖入「应用程序」）
2. 从启动台打开 FileNest
3. 若 macOS 阻止运行，请到 **系统设置 → 隐私与安全性** 中点击 **仍要打开**

## 功能

- **文件夹库** — 拖入常用目录，集中管理
- **标签** — 按项目分类（如 Work、Website、AI Creation）
- **收藏** — 固定最常打开的文件夹
- **菜单栏** — 不切换应用即可快速进入
- **最近记录** — 快速回到最近打开的目录
- **备份** — JSON 导入导出，数据留在本机

## 系统要求

- macOS 13 Ventura 或更高版本
- Apple 芯片或 Intel Mac
